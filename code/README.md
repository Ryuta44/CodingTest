# README

## **フォルダ構成**

フォルダ構成は以下のようになっている。

```jsx
└── code
     ├──Results
     │     ├──result1.txt
     │     ├──result2.txt
     │     ├──result3.txt
     │     └──result4.txt
     ├──TestCase
     │     ├──create_testcase.py
     │     ├──test.txt
     │     ├──test1.txt
     │     ├──test2.txt
     │     ├──test3.txt
     │     └──test4.txt
     ├──Q1.py
     ├──Q2.py
     ├──Q3.py
     └──Q4.py
```

各課題の監視ログデータ

```jsx
code/TestCase
```

各課題の出力結果

```jsx
code/Results
```

## **使い方**

初めに以下のコマンドでgithubからソースコードをダンロードする。

```jsx
git clone https://github.com/
```

### 課題の実行方法

課題1

```jsx
python3 code/Q1.py
```

以降の課題も"Qx.py"のxを課題に対応する番号を入れて実行する。

またコードの解説は"Qx.txt"に記載されている。
